# Disease Prediction System

## Overview
This project predicts whether a patient is likely to have diabetes using Machine Learning algorithms and medical data.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

---

## Algorithms Used
- Logistic Regression
- SVM
- Random Forest

---

## Dataset
Pima Indians Diabetes Dataset

Features used:
- Glucose
- Blood Pressure
- BMI
- Insulin
- Age
- Pregnancies

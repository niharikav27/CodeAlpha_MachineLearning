# 💳 Credit Scoring Model

## 📌 Overview

Built a machine learning model to predict whether a person will default on a loan using financial data.

---

## 🎯 Objective

Predict an individual's creditworthiness using past financial data.
Classify customers as:

* 0 → No Default
* 1 → Default

---

## 🔍 Work Done

* Performed EDA (checked missing values, data distribution)
* Handled missing data using median
* Created new feature: Debt-Income Ratio
* Applied feature scaling

---

## 🤖 Models Used

* Logistic Regression
* Random Forest

---

## 📊 Evaluation

Used:

* Precision
* Recall
* F1-Score
* ROC-AUC

---

## 📈 Result

* Random Forest performed better
* Recall was prioritized to reduce risk of missing defaulters
